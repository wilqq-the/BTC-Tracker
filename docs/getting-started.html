Docker Installation

Run anywhere with Docker containers

Quick Start

docker pull docker.io/thewilqq/btc-tracker:latest
docker run -d -p 3000:3000 -v ./data:/app/src/data docker.io/thewilqq/btc-tracker:latest

This starts the application on localhost:3000 with data stored in ./data

Docker Compose (Recommended)

Create a docker-compose.yml file:

services:
  btc-tracker:
    image: docker.io/thewilqq/btc-tracker:latest
    container_name: ${CONTAINER_NAME:-btc-tracker}
    ports:
      - "${PORT:-3000}:3000"
    volumes:
      - ${DATA_PATH:-./data}:/app/src/data
    restart: unless-stopped
    environment:
      - NODE_ENV=${NODE_ENV:-development}
      - PORT=3000

Then start the application:

docker-compose up -d

Configuration Options

CONTAINER_NAME Container name (default: btc-tracker)
PORT Application port (default: 3000)
DATA_PATH Data storage path (default: ./data)
NODE_ENV Environment (default: development)

Manual Installation

Direct installation for developers

Prerequisites

  • Node.js (v14 or higher)
  • npm or yarn package manager
  • Git (for cloning repository)

Installation Steps

# Clone the repository
git clone https://github.com/wilqq-the/btc-tracker.git
cd btc-tracker

# Install dependencies
npm install

# Start the application
npm start

The application will be available at http://localhost:3000
