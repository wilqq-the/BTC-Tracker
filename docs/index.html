About BTC Tracker

BTC Tracker is a self-hosted web application that helps you track your Bitcoin transactions and monitor the performance of your investment. It's designed to be simple to use, secure, and privacy-focused.

With BTC Tracker, you can manage your Bitcoin transactions, see the current price of Bitcoin, calculate your profit/loss, view historical performance, and export your data for tax reporting or backup purposes.

Quick Start

You can run BTC Tracker using Docker for a quick and easy setup:

docker pull docker.io/thewilqq/btc-tracker:latest
docker run -d -p 3000:3000 -v ./data:/app/src/data docker.io/thewilqq/btc-tracker:latest
