BTC Tracker

A simple and powerful Bitcoin transaction tracker for monitoring your investment

About BTC Tracker

BTC Tracker is a self-hosted web application that helps you track your Bitcoin transactions and monitor the performance of your investment. It's designed to be simple to use, secure, and privacy-focused.

With BTC Tracker, you can manage your Bitcoin transactions, see the current price of Bitcoin, calculate your profit/loss, view historical performance, and export your data for tax reporting or backup purposes.

The BTC Tracker dashboard showing transaction history and performance metrics

Key Features

Transaction Management

Add, edit, and delete Bitcoin transactions with support for multiple currencies.

Real-time Price Updates

Get up-to-date Bitcoin prices in EUR and USD from CoinGecko.

Performance Tracking

Monitor your investment performance with profit/loss calculations and percentages.

Historical Data

View historical Bitcoin price data to analyze market trends.

CSV Import/Export

Import transactions from CSV files or export your data for backup or analysis.

Multi-currency Support

Support for multiple currencies with automatic exchange rate conversion.

Getting Started

You can run BTC Tracker using Docker for a quick and easy setup:

docker pull docker.io/thewilqq/btc-tracker:latest
docker run -d -p 3000:3000 -v ./data:/app/src/data docker.io/thewilqq/btc-tracker:latest

Or use Docker Compose for more configuration options:

services:
  btc-tracker:
    image: docker.io/thewilqq/btc-tracker:latest
    container_name: btc-tracker
    ports:
      - "3000:3000"
    volumes:
      - ./data:/app/src/data
    restart: unless-stopped
    environment:
      - NODE_ENV=production
      - PORT=3000
