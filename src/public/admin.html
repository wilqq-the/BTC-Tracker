CSV Format

Your CSV file should have the following columns:

  • Date: Transaction date in YYYY-MM-DD format
  • Type: "buy" or "sell"
  • Amount (BTC): BTC amount bought or sold
  • Exchange: Exchange name or "manual" for manual entries
  • Original Currency: Currency code (EUR, USD, PLN, etc.)
  • Original Price: Price per BTC in the original currency
  • Original Cost: Total cost in the original currency
  • Original Fee: Transaction fee in the original currency

Tip: Download the CSV Template for a properly formatted example.
